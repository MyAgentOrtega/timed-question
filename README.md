# Timed-Question   <br/>

### <u> Description: </u>
This is a simple math quiz that contains 3 questions. When you answer incorrectly it deducts 10 seconds. Total seconds left is the total score.

### <u> Notes: </u>
Initials button is shown at end of 3 question quiz.
